# TimeCue Releases

Public binary distribution and automatic-update channel for **TimeCue**, a desktop LTC/MTC cue controller for Resolume Arena.

This repository contains signed release packages and update metadata only. The application source and project repository remain private.

## Downloads

Open [Releases](https://github.com/4H1Mzs/TimeCue-Releases/releases) to download the latest supported macOS and Linux packages. Installed builds use the same releases for in-app update checks.

Windows automatic updates will be added after the installer has an Authenticode signing identity.
